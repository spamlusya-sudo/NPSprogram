# Готовый пакет для Timeweb Cloud

Сайт уже подготовлен для публикации в **Timeweb Cloud** с запросом логина и пароля через стандартное окно браузера.

## Данные для входа
- Логин: `admin`
- Пароль: `ppro127`

## Как выложить в Timeweb Cloud

### Вариант через Apps / Dockerfile
1. Создайте приложение в **Timeweb Cloud**.
2. Выберите деплой из **Dockerfile**.
3. Загрузите содержимое этой папки или архив в репозиторий/источник, который будет использовать Timeweb.
4. Порт приложения: `8080`.
5. После сборки опубликуйте приложение наружу.
6. Привяжите домен или используйте выданный Timeweb адрес.
7. Включите HTTPS/SSL в панели Timeweb.

### Локальная проверка перед публикацией
```bash
docker build -t azs-dashboard-timeweb .
docker run -p 8080:8080 azs-dashboard-timeweb
```

Потом откройте `http://localhost:8080`.

## Что внутри
- `index.html` — ваш сайт
- `nginx.conf` — конфигурация Nginx
- `.htpasswd` — уже настроенный пароль
- `Dockerfile` — контейнер для деплоя в Timeweb Cloud

## Важно
- Для реального использования публикуйте сайт только по **HTTPS**.
- После первого запуска лучше сменить пароль на более длинный.
