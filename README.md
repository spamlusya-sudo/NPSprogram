# NPSprogram — версия с общим сохранением и расширенной аналитикой

Что изменено:
- отдельные минимальные уровни для АИ-95, АИ-92 и ДТ;
- на карточках остатков при уходе ниже минимума показываются дни до полного истощения;
- графики на вкладках «Нефтебаза» и «Прогноз остатков» умеют показывать 1 месяц, 2 недели и 1 неделю;
- на графиках можно переключать режимы «История» / «Прогноз» и выбирать отдельный вид топлива;
- у поставок добавлена цена закупки;
- завершённые поставки автоматически попадают в исторический блок;
- в оптовой реализации добавлен расчёт доступного к продаже объёма по каждому виду топлива с учётом будущих поставок и ежедневного расхода;
- маржа считается отдельно по АИ-95, АИ-92 и ДТ;
- добавлена вкладка «Финансы и KPI» с оборачиваемостью и ключевыми показателями по запасам.

## Как обновить сайт

1. Замените файлы в GitHub-репозитории содержимым этой папки.
2. Убедитесь, что в корне репозитория лежат:
   - `index.html`
   - `package.json`
   - `netlify.toml`
   - папка `netlify/functions`
3. В Netlify откройте ваш сайт.
4. Нажмите **Deploys → Trigger deploy → Deploy site**.
5. После нового деплоя откройте сайт и обновите вкладки браузера.

## Что важно знать

- История с 01.01.2026 строится по сохранённым ежедневным снимкам и операциям. Чем дольше сайт работает в новой версии, тем точнее фактическая история.
- Если одновременно редактировать сайт в нескольких старых вкладках, сервер оставит более новую версию данных.
